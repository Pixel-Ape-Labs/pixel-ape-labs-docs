# View Your Solana Wallets

This command will return a list of the wallets that you have linked to your profile on Pixel Ape Labs.

:::info

Default Permissions: **Everyone**  
Command Type: **Slash Command**

:::

## Usage Example

### Slash Command

```
/wallets
```
