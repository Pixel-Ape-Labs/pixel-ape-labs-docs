# Rewards

This command will show you any active rewards that still need to be claimed. If there are any rewards, you will be giving a link to go claim them.

:::info

Default Permissions: **Everyone**  
Command Type: **Slash Command**

:::

## Usage Example

### Slash Command

```
/rewards
```
